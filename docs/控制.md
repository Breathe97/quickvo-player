## 控制

### 设置渲染模式

- 使用 cdn 原画还是通过 sei 剪切后的视频流渲染。

```ts
player.setDisplayMode = (mode: 'original' | 'cut')
```

### 开始播放

- 传入 cdn 地址即可

```ts
player.start = async (url: string)
```

### 设置静音

- 控制视频是否静音

```ts
player.setMute = (state?: boolean)
```

### 设置扬声器音量

- 控制视频音量大小

```ts
player.setOutputGain = (gain: number)
```

### 获取原画流

- 获取 cdn 直接转码后的视频流 (剪切流通过回调获取)

```ts
player.getStream()
```

### 停止播放

- 传入 cdn 地址即可

```ts
player.stop()
```
