## 回调相关

### 用户回调

- 当切换为 cut 模式时 使用 SEI 信息 分割视频后输出,其中包含剪切后的完整用户信息和对应的媒体流。

```ts
player.on.users = (e) => {
  console.log('\x1b[38;2;0;151;255m%c%s\x1b[0m', 'color:#0097ff;', `------->Breathe: e`, e)

  const [user] = users
  // 以单个用户渲染为例
  // 拿到所需要的 stream 然后进行渲染 (这里以原生js使用方式作为参考)
  document.querySelector<HTMLVideoElement>('#my-dom-view')?.srcObject = user.mc_video.stream
}
```

### 用户信息说明

```ts
interface AudioInfo {
  isEnable: boolean // 是否开启设备
  updateTime: number // 更新时间
  isMute: boolean // 是否静音
  volume: number // 音量值
}

interface VideoInfo {
  isEnable: boolean // 是否开启设备
  updateTime: number // 更新时间
  stream?: MediaStream // 用户流
}

mc_audio: AudioInfo | undefined // 麦克风
mc_video: VideoInfo | undefined // 摄像头

ss_audio: AudioInfo | undefined // 屏幕共享视频
ss_video: VideoInfo | undefined // 屏幕共享音频
```
